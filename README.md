# 📓 binder-rich-editor

## ⬆️ Direções

1. Clone o repo: `git clone https://github.com/marcelovicentegc/binder-rich-editor.git`
2. cd nele: `cd binder-rich-editor`
3. Instale as dependências (use `npm`): `npm i`
4. Rode a aplicação: `npm start`
